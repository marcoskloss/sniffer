# Packet Sniffer

A packet sniffer application built with C# and Windows Forms for capturing and analyzing network traffic in real time.

The project provides a graphical interface to monitor network packets, inspect communication details, and visualize network interface usage.

## Legacy C++ version

Packet Sniffer was originally developed as a command-line C++ application. The project has since been redesigned and migrated to C#, introducing a modern graphical interface and an improved user experience.

The legacy C++ implementation is still available for reference under the `legacy/cpp` directory.

## Technologies

- C#
- .NET
- Windows Forms
- SharpPcap
- PacketDotNet
